# Football Media Network — Public Website

Public marketing site for Football Media Network: an automated football media
outlet that publishes live goal alerts, AI match reports, transfer news, and
Man of the Match breakdowns to Telegram, X, TikTok, and YouTube.

Static site, no build step. Deployed via Vercel.

- `index.html` — landing page
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Service
- `assets/samples/` — sample output graphics
